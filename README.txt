Project Name: FBG-AutoPost
Contributor: Durru Aravind
Date: 30 December, 2021
Description: A mini project based on web automation using Python libraries - Selenium, Tkinter and Webdriver. This project automates facebook group(s) posting though a GUI and Chromedriver.
Pre-requisites:

1. Install Chromedriver and them run the below command to Install all packages in "requirements.txt":
    pip install -r requirements.txt
